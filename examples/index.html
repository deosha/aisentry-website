Generate Your Own Reports

Try scanning any project to generate reports in multiple formats:

# HTML report (interactive, with audit)
ai-security-cli scan ./your-project -o html -f report.html

# JSON report (machine-readable)
ai-security-cli scan ./your-project -o json -f report.json

# SARIF report (GitHub Code Scanning)
ai-security-cli scan ./your-project -o sarif -f report.sarif
